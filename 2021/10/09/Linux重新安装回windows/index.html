Linux 重新安装回 Windows

1、制作U盘启动盘(Windows: rufus)

2、重新格式化磁盘

2.1 进入引导界面，并进入命令提示符

2.2 输入diskpart，进入DISKPART命令模式；

2.3 输入list disk，列出当前磁盘信息；

2.4 输入select disk 0，选择磁盘0；

2.5 输入clean，删除磁盘分区；

2.6 输入convert gpt，将磁盘转换为GPT；

2.7 输入create partition efi size=320(单位: M)，创建EFI

2.8 输入create partition msr size=512(单位: M)，创建Misrosoft保留分区

2.9 输入create partition primary size = 102400(单位: M)，创建100G的主分区，1G=1024M；这里的主分区相当于我们的C盘，大小可以随意指定，剩下的硬盘空间在安装完系统之后，进入系统在磁盘管理中可以自行分配；

2.10 输入format fs=ntfs quick回车，格式化主分区；windows磁盘的文件系统一般都是NTFS格式，所以这里我们要格式化为NTFS格式。看到显示【成功格式化该卷】的时候说明操作成功了。

3、重新进入引导界面
