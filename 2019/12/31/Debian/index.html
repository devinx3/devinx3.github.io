使用说明

1、若需要使用https源，则需要执行sudo apt-get install apt-transport-https ca-certificates安装https源

2、备份配置文件：

sudo cp -a /etc/apt/sources.list /etc/apt/sources.list.bak

3、修改sources.list文件，将http://deb.debian.org替换成https://repo.huaweicloud.com，同时也将http://security.debian.org替换成https://repo.huaweicloud.com (可修改为上面的网址)，可以参考如下命令：

4、执行apt-get update更新索引
